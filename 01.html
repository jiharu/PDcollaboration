Each pixel (picture element) contains a color value for a number of channels – usually red, green, and blue. When you view these channels together, at a high enough resolution, all of the different colored pixels form a complete image. These images can then be output to video, film, print, etc. **Vector** images are not based on pixel patterns, but instead use mathematical formulas to draw lines and curves that can be combined to create an image from geometric objects such as lines, circles, triangles, squares, and other polygons. Vector images are edited by manipulating the lines and curves that make up the image using a program such as Adobe Illustrator. Bitmap formats are best for images that need to have a wide range of color gradations, such as most photographs. Vector formats, on the other hand, are better for images that consist of a few areas of solid color. Examples of images that are well suited for the vector format include logos and text. To convert a 2D vector image into 3D using Blender your source image must be an .svg file. Black and white silhouette designs work best. ### How: - Prepare .svg silhouette vector image. - From Blender, import the image. ![2d3d shape bl00](img/07/Chandelier.svg) The source vector image as .svg. ![2d3d shape bl](img/07/bl01.png) - Choose the imported object and scale it up. - From the right tool settings, choose the curve and add the Extrude value. It will define the height of the geometry. Then you can export the geometry with several different formats including .obj and .stl. ![2d3d shape bl01](img/07/bl02.png) ### Tutorial - [Blender-Converting 2D Image to 3D Object by bugzilla2001](https://www.youtube.com/watch?v=ows2QTiMRPg)
