# Class 3: # Practice ### Drawing on 3D surface How to do: **Preparation** - Select a base object and from the mouse right button menu, apply Make live. - You can find a base object from Window / General Editors / Visor. In the Visor, under the Sculpting Base Meshes, there are several folders which have base meshes. You can select one and click the mouse right button then click the import Maya file pop-up menu. - [Set a live surface](https://knowledge.autodesk.com/support/maya/learn-explore/caas/CloudHelp/cloudhelp/2016/ENU/Maya/files/GUID-EBF1E652-4DDD-423F-B42E-3BF85AE46A5D-htm.html) **Drawing on surface** - Select Mesh Tools / Create Polygons and draw on the surface. Once you finish, press enter to complete it. - To make it as a stable polygon, apply Mesh / Triangulate - To add thickness, select all faces (in the component mode) apply Edit Mesh / Extrude. and add a thickness value to Local Translate Z attribute. **Subdivision** - To make it smoother and more organic, select the object and then apply Mesh / Smooth. ![2d3d_maya_01](img/07/mask.png) ![2d3d_maya_03](img/07/maya_live02.png) ![2d3d_maya_04](img/07/maya_live03.png) ### Constrain objects on a 3D surface How: - Related tutorial: [Normal (Constraint) #example_01](https://www.youtube.com/watch?v=ZsAiX6Xyf6A) - Select the constraining object and then shift-select the constrained object, and apply Animation module set // Contrain / Geometry constrain. - Repeat the selection of constraining and constrained objects, and then apply Animation module set // Constrain / Normal constrain. - You can change the constrained object's direction by its normal Constraint node / Normal Constraint Attributes. Change the Aim vector to change the direction. (1: on, 0:off) - If you want to duplicate the constrained object with its attributes, you can go Edit / Duplicate Special and check Duplicate Input Graph and duplicate your objects. They will behave the same as the previous objects. ![2d3d_maya_02](img/07/maya_live01.png) ### Helpful references - [Maya: Troubleshoot imported vector data](http://help.autodesk.com/view/MAYAUL/2016/ENU//index.html?guid=GUID-05A64113-C6EE-4456-88A4-2ADCF23DD40E) - [Maya: Import vector data as curves or generate polygonal surfaces](https://knowledge.autodesk.com/support/maya-lt/learn-explore/caas/CloudHelp/cloudhelp/2016/ENU/MayaLT/files/GUID-EF3ED536-7C44-4225-AD4F-2FB0E0DD0362-htm.html) - [From 2D to 3D Using Inkscape to Shapeways / OpenSCAD) by unkerjay in 3D-Printing](http://www.instructables.com/id/From-2D-to-3D-Using-Inkscape-to-Shapeways-OpenSCAD/) ______________________ ## Group Assignment
