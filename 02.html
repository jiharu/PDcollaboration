Low quality is free but high quality stl data is not free from Selva3D. ## 2D gray image to 3D using Thinkercad - [Thinkercad](https://www.tinkercad.com/) The source image is .svg (a kind of vector graphic), and Thinkercad turns it into 3D geometry immediately. ![2d3d shape tc](img/07/thinkercad.png) The dialog window pops up when you import an svg file. ![2d3d shape tc01](img/07/thinkercad01.png) The dialog window pops up when you export the transformed geometry. ------------------------------------ ## Using Blender ### Source: vector image. To convert a 2D vector image into 3D using Blender your source image must be an .svg file. Black and white silhouette designs work best. ### How: - Prepare .svg silhouette vector image. - From Blender, import the image. ![2d3d shape bl00](img/07/Chandelier.svg) The source vector image as .svg. ![2d3d shape bl](img/07/bl01.png) - Choose the imported object and scale it up. - From the right tool settings, choose the curve and add the Extrude value. It will define the height of the geometry. Then you can export the geometry with several different formats including .obj and .stl. ![2d3d shape bl01](img/07/bl02.png) ### Tutorial - [Blender-Converting 2D Image to 3D Object by bugzilla2001](https://www.youtube.com/watch?v=ows2QTiMRPg) ---------------------------------------- ## Using Maya ** The source can be a vector or bitmap image.** ### For vector images use Illustrator version 8 .ai or .eps files - [Quick Tip: Converting Vector Logos to 3D from Illustrator to Maya](https://cgi.tutsplus.com/tutorials/quick-tip-converting-vector-logos-to-3d-from-illustrator-to-maya--cg-6466): This has two tutorials. One to extrude and the other using Bevel Plus. ### Using bitmap image such as .jpg or .png and more. - [Maya Tutorial- how to create 3D logo's from pictures](https://www.youtube.com/watch?v=WxlLUDZQS6E) ![2d3d_maya_con](img/07/maya_convert.png) ### Drawing on 3D surface How to do: **Preparation** - Select a base object and from the mouse right button menu, apply Make live. - You can find a base object from Window / General Editors / Visor. In the Visor, under the Sculpting Base Meshes, there are several folders which have base meshes. You can select one and click the mouse right button then click the import Maya file pop-up menu. - [Set a live surface](https://knowledge.autodesk.com/support/maya/learn-explore/caas/CloudHelp/cloudhelp/2016/ENU/Maya/files/GUID-EBF1E652-4DDD-423F-B42E-3BF85AE46A5D-htm.html) **Drawing on surface** - Select Mesh Tools / Create Polygons and draw on the surface. Once you finish, press enter to complete it. - To make it as a stable polygon, apply Mesh / Triangulate - To add thickness, select all faces (in the component mode) apply Edit Mesh / Extrude. and add a thickness value to Local Translate Z attribute. **Subdivision** - To make it smoother and more organic, select the object and then apply Mesh / Smooth. ![2d3d_maya_01](img/07/mask.png) ![2d3d_maya_03](img/07/maya_live02.png) ![2d3d_maya_04](img/07/maya_live03.png) ### Constrain objects on a 3D surface How: - Related tutorial: [Normal (Constraint) #example_01](https://www.youtube.com/watch?v=ZsAiX6Xyf6A) - Select the constraining object and then shift-select the constrained object, and apply Animation module set // Contrain / Geometry constrain. - Repeat the selection of constraining and constrained objects, and then apply Animation module set // Constrain / Normal constrain. - You can change the constrained object's direction by its normal Constraint node / Normal Constraint Attributes. Change the Aim vector to change the direction. (1: on, 0:off) - If you want to duplicate the constrained object with its attributes, you can go Edit / Duplicate Special and check Duplicate Input Graph and duplicate your objects. They will behave the same as the previous objects. ![2d3d_maya_02](img/07/maya_live01.png) ### Helpful references - [Maya: Troubleshoot imported vector data](http://help.autodesk.com/view/MAYAUL/2016/ENU//index.html?guid=GUID-05A64113-C6EE-4456-88A4-2ADCF23DD40E) - [Maya: Import vector data as curves or generate polygonal surfaces](https://knowledge.autodesk.com/support/maya-lt/learn-explore/caas/CloudHelp/cloudhelp/2016/ENU/MayaLT/files/GUID-EF3ED536-7C44-4225-AD4F-2FB0E0DD0362-htm.html) - [From 2D to 3D Using Inkscape to Shapeways / OpenSCAD) by unkerjay in 3D-Printing](http://www.instructables.com/id/From-2D-to-3D-Using-Inkscape-to-Shapeways-OpenSCAD/) ______________________ ## Group Assignment
